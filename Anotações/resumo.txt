# Resumo:

## O que é Git?
### É um sistema de versionamento de código distribuído (ajuda a criar e monitorar diferentes versões do código).

## CLI:
### Interface de Linha de Comando (permite que os usuários digitem comandos manuais para executar uma tarefa).
## GUI:
### Interface Gráfica do Usuário (utiliza recursos visuais como botões, ícones gráficos, imagens, etc. para interagir com o sistema).

## SHA1:
### Secure Hash Algorithm (é um conjunto de funções hash criptográficas projetadas pela NSA - Agência de Segurança Nacional dos EUA).
